# Изображения

Папка для архивных фото и сканов газет.

## Рекомендуемые файлы (когда появятся)

| Файл | Назначение |
|------|------------|
| `teterev-portrait.jpg` | Портрет тренера для hero |
| `gym-archive.jpg` | Спортзал, фон секций |
| `g-01-scan.jpg` | Скан статьи G-01 |
| `g-02-scan.jpg` | Скан статьи G-02 |

## Правила

- Оригиналы хранить отдельно (см. `/newspapers/raw/` в SPECIFICATION.md).
- Web-версия: JPEG, max 2000px, без агрессивной ретуши.
- Путь в HTML: `../assets/images/имя-файла.jpg`

На MVP сайт работает без изображений — используется CSS-фактура «архивной бумаги».
