# Поля форм «Прислать материал»

> Спецификация полей. На MVP- сайте формы визуальные, без отправки на сервер.

## Форма 1 — Прислать воспоминание

| Поле | Тип | Обязательное |
|------|-----|--------------|
| Имя | text | да |
| Контакт (телефон / Telegram) | text | да |
| Кем вы были связаны с Михаилом Тетеревым | select + text | да |
| Текст воспоминания | textarea | да |
| Можно ли публиковать | radio: да / нет / анонимно | да |
| Можно ли связаться для интервью | radio: да / нет | да |
