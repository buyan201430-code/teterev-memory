## Правила обновления

1. **Новая газета** → все новые имена → строка в этой таблице + пробел в [`gaps.md`](gaps.md).
2. **Человек откликнулся** → статус `найден`, контакт в `people/contacts.csv` (не в Git).
3. **Интервью проведено** → создать `INT-...`, статус `интервью проведено`.
4. **Карточка STU- готова** → статус `карточка готова`, ссылка в колонке «Карточка».
5. **Имя оказалось ошибкой** → статус `ошибка`, примечание, не удалять строку.
