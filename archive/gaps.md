# Пробелы архива

> Реестр того, чего не хватает. Обновлять при каждой новой находке.  
> Закрытый пробел → статус `resolved`, ссылка на карточку, которая закрыла пробел.

## Как пользоваться

1. Каждый пробел — строка с ID `GAP-NNN`.
2. При закрытии — не удалять строку, менять статус на `resolved`.
3. Новый пробел из газеты / интервью → добавить сразу, не откладывать.
4. Ссылка на этот файл — со страницы `/looking-for` (блок «Что мы ещё не знаем» — опционально, позже).
