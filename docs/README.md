# Документация сайта «Школа силы»

MVP статического архива для проекта книги о Михаиле Михайловиче Тетереве.

## Структура

```
/site          — HTML, CSS, JS (то, что открывается в браузере)
/content       — исходники текстов (markdown)
/assets/images — место для фото и сканов
/docs          — эта документация
```

## Как открыть локально

### Способ 1 — двойной клик (самый простой)

Откройте файл:

```
site/index.html
```

Работает без сервера. Все тексты встроены в HTML.

### Способ 2 — локальный сервер (опционально)

Из корня проекта:

```bash
cd site && python3 -m http.server 8080
```

Откройте: http://localhost:8080
